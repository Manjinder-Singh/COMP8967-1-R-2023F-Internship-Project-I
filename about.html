About Climate Neutral

Our Mission

Welcome to "Climate Neutral," where we are dedicated to empowering users in making informed decisions related to climate change and environmental sustainability. Our mission is to provide an innovative Multi-Objective Decision-Making (MODM) web application to help you evaluate and select optimal solutions considering various critical factors.

Key Features

  • Authentication: We offer user authentication to ensure that authorized users can access our application.
  • Dynamic Input: Users can create and customize category-scenario combinations, offering a flexible and adaptable environment.
  • Save and Import: Save your input values as text files for future reference and easily import them to reproduce previous results.
  • Visualization: Our application performs MODM analysis and presents results through interactive charts.
  • Category Information: Customize each category with a name, weight, impact direction, and evaluation type.
  • Scenario Information: Define scenario names and values for each category, whether numerical or descriptive.
  • Desktop Optimization: Our design is optimized for desktop screens, ensuring a seamless user experience.
  • Local Storage: We save your data in browser local storage, making it easy to retrieve your last session.
  • Data Initialization: Our application checks for previously stored data in local storage and provides an empty table for new input.

How MODM Works

  1. Conversion of Descriptive Categories: Descriptive category values are converted into numerical equivalents for consistency.
  2. Normalization of Scenario Values: Scenario values are normalized based on category direction.
  3. Scenario Ranking: The application calculates scenario scores and ranks them, helping you identify the most favorable options.
